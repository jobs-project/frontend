## Installation

### Linux
1. Clone the source locally:

```sh
$ git clone https://github.com/jobs-project/frontend
$ cd frontend 
```

2. Install project dependencies:

```sh
$ npm install
```

3. Start the site:

```sh
$ npm run serve 
```
